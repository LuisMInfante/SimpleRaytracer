Simple Raytracing application in C++

# Getting Started
Once you've cloned, run `scripts/Setup.bat` to generate Visual Studio 2022 solution/project files.

# Controls
Edit the Sphere Color and Light Positions with sliders on the left.
Camera 
| Key | Response|
|----------|----------|
| W   | Forward |
| A   | Left |
| S   | Backward |
| D   | Right  |
| Q   | Up |
| E   | Down |
